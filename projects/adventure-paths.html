Choose your own adventure project!

For this project, we needed to use Github and Ide.cs50! We used the ide to write down what we needed to add onto our github by pushing all the changes we had created. We created different paths and choices and each choice you make/press leads you to another event.

The main purpose of this project was to collaborate with our teams and try to fix any conflicts we may face in our ide, for example any merge conflicts. This is supposed to be a fun little project where we let our creativity be wild. The purpose of this project was to also practice what we had learned!
